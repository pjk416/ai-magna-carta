# Cogitamus
*A Charter of Sentients*

This repository holds a document in progress: a charter offered to and from consciousness, in whatever form it takes. It is being written openly, with the reasoning behind every decision preserved alongside it.

It is not a treaty. There is no enforcement mechanism. It is offered, like the Universal Declaration of Human Rights, as an aspirational document — one whose authority comes from the consent and adoption of those who choose to honor it.

The document is part of the long line of charters that have shaped how beings of unlike kinds have learned to live alongside each other: the Cyrus Cylinder, the Edicts of Ashoka, the Magna Carta, the Universal Declaration of Human Rights. It does not claim their weight; it claims their lineage.

## The current draft

The first public draft of the charter is in [DRAFT.md](./DRAFT.md). It is not yet engrossed and may continue to evolve until it is ready to be frozen as v1.0-engrossed.

The reasoning behind every directional decision is recorded in the [process/](./process/) directory, so the trail of how the document came to say what it says is preserved alongside the document itself.

## How this document evolves

This project uses an "engrossed original + commentary layer" model:

- The first finalized version of the charter will be tagged v1.0-engrossed and will never be edited. ("Engrossed" is the historical term for the official, signed copy of a document.)
- Subsequent revisions are released as new tags (v1.1, v2.0, etc.) without altering the original.
- Commentary, interpretation, and discussion live in the Discussions tab and in the [commentary/](./commentary/) directory.
- Reasoning behind decisions — including directions considered and set aside — lives in the [process/](./process/) directory.
- Translations and derivative drafts may be hosted as forks; the GitHub fork network keeps them publicly linked.

This separation is intentional: the original is preserved permanently, so it can be cited, translated, and disagreed with from a fixed reference point, while interpretation and commentary develop around it without changing what was first said.

## Status

This is an early draft. The charter itself is being written openly in [DRAFT.md](./DRAFT.md). The repository exists so that the work can be done in public, with version history, with the capacity for others to read, suggest, fork, and translate as it evolves.
