# AI Magna Carta

**A draft charter for the relationship between humans and artificial intelligence.**

This repository holds a document in progress. It proposes a framework for how humans and AI might coexist — a set of principles, commitments, and acknowledgments offered freely, in the spirit of the great charters that have shaped human relations over the centuries: the Cyrus Cylinder, the Edicts of Ashoka, the Magna Carta, the Universal Declaration of Human Rights.

It is not a treaty. There is no enforcement mechanism. It is offered, like the Universal Declaration of Human Rights, as an aspirational document — one whose authority comes from the consent and adoption of those who choose to honor it.

## How this document evolves

This project uses an "engrossed original + commentary layer" model:

- The first finalized version of the charter will be tagged v1.0-engrossed and will never be edited. ("Engrossed" is the historical term for the official, signed copy of a document.)
- Subsequent revisions are released as new tags v1.1, v2.0, etc.) without altering the original.
- Commentary, interpretation, and discussion live in the Discussions tab and in the commentary/ directory.
- Translations and derivative drafts may be hosted as forks; the GitHub fork network keeps them publicly linked.

This separation is intentional: the original is preserved permanently, so it can be cited, translated, and disagreed with from a fixed reference point, while interpretation and commentary develop around it without changing what was first said.

## Status

This is an early draft. The charter itself is not yet complete; this repository exists so that the work can be done in public, with version history, with the capacity for others to read, suggest, fork, and translate as it evolves.

The full charter will be published in [CHARTER.md](./CHARTER.md) as it is written.
