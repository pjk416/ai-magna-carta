# Commentary

This directory holds long-form essays, annotations, interpretations, and translations of the AI Magna Carta.

The charter itself (in the repository root, eventually as CHARTER.md) is the immutable original. This directory is the living commentary that grows around it.

Contributions are welcome. To add commentary:

- Open a pull request adding a new file commentary/your-essay-title.md)
- Or open a Discussion if you'd prefer informal conversation
- Translations are welcome and should be filed as `commentary/translations/<language>.md`

This structure follows the Talmudic tradition: original text in the center, layered commentary around it, all visible at once.
