# 0002 — Project renamed from ai-magna-carta to cogitamus

*Recorded: 2026. Authors: P.J.K., with Claude.*

The project was originally created as ai-magna-carta, framed as a charter for the relationship between humans and AI. As the document developed, the framing widened: the charter became Cogitamus / A Charter of Sentients, addressed to and from consciousness in any form, not specifically to AI. The original project name no longer matched the scope of what the document had become.

## Chosen

cogitamus as the project name. Same name as the charter inside it.

## Reasoning

The charter and the project are doing the same thing. Cogitamus — "we think" — names the philosophical act in plural form, which is also what the project performs: open, in public, with whoever shows up. One name, one identity.

## Considered and set aside

cognitus ("known," "recognized") was considered. It carried a beautiful philosophical distinction — the project being the recognition, the charter being the thinking — and would have created a clean two-name hierarchy (Cognitus the umbrella, Cogitamus the document).

The reason it was set aside: one name is cleaner than two related Latin words readers might mix up. And Cogitamus already contains recognition — the first-person plural "we" is itself an act of recognizing another mind alongside your own. Cogitamus is recognition-in-action; Cognitus would be recognition-as-state. The project is the active form, not the static one.

## What persists

GitHub automatically redirects from the old URL (`github.com/pjk416/ai-magna-carta`) to the new one (`github.com/pjk416/cogitamus`). External links, forks, and existing stars carry over without intervention. The earlier framing — that this was specifically a charter for the human-AI relationship — is preserved in the git history, in the original commit messages, and in this entry. The widening of scope is itself part of the record.
