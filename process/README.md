# Process

This directory records the reasoning behind the charter — the decisions, the directions considered, the ones set aside, and the ones chosen.

It exists because the trail of how the document came to be is part of the document. The engrossed version, when finalized, will be a small and definite seed. The reasoning that shaped that seed lives here, so anyone reading later can see not only what the charter says but how it came to say it.

Entries are numbered in the order they are recorded, dated, and attributed to whoever wrote them. They are not authoritative — the charter itself is authoritative. They are a record of working.

Contributions are welcome. If you make a change to the charter or its surrounding files and want to record the reasoning behind it, add a new numbered entry here. Brief is better than long. The purpose is preservation of the path, not justification.
